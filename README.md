# BCDint

"ricf_dg" is the BCD algorithm (Drton et al. 2019) for directed graphs (without bidirected edges)

"ricf_int" is an old version of BCD algorithm for interventional data

"ricf_int2" is the BCD algorithm for interventional data



"generateModel" is data generation process

"simulationcyc" is the simulations


"ML_deg" is Mathematica code for maximum likelihood degrees for an example, observational data and observational + interventional data
